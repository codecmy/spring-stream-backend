User manual (Chrome):
1. Start backend and make sure FFmpeg is installed and available in system PATH.
2. Open this page from an HTTP origin (not file://) so cookies and CORS work correctly.
3. Paste your Hanko API URL and click Initialize Widget.
4. Register/login in the Hanko widget, then click Validate Session.
5. Once auth status becomes valid, use upload/list/play controls.
6. For Chrome, HLS will play through hls.js fallback automatically.
7. Use Logout to end session and re-lock API controls.
